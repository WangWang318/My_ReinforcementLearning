# 一、环境安装

🍌

## 1、安装超级马里奥环境

```
https://pypi.org/project/gym-super-mario-bros/
```

🍑

```
pip install gym-super-mario-bros
```

🍑

## 2、安装 python 版本的 nes 模拟器

```
https://pypi.org/project/nes-py/

https://github.com/Kautenja/nes-py
```

🍍

```
pip install nes-py
```

🍍
